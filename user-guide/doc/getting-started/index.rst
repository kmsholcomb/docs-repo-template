===============
Getting started
===============

.. Define |product name| in conf.py

Include any tasks that are needed to get the product to a usable state.
Examples can include (but are not limited to):

- Accessing the product - links to product, how to log in, how to
  request access

- Navigating - overview of the UI (if any)

- Configuring - configuration task to get the product to a usable state

- Authenticating - how to authenticate the user or product

Add all getting stated task filenames to the TOC list.


This section provides information about getting started using the
|product name| product.
